# Aether Icons

All icons are provided as a single SVG sprite file at
`luci-static/aether/icons/lucide.svg`.

## Usage

```html
<svg class="ae-icon"><use href="/luci-static/aether/icons/lucide.svg#icon-cpu"/></svg>
```

The sprite contains a curated subset of the Lucide icon set, drawn with a
consistent 1.75 stroke width and 24×24 viewBox. New icons can be appended
by copying the corresponding `<symbol>` definition from
[lucide.dev](https://lucide.dev).
